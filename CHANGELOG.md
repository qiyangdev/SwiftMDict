# Changelog

## 0.1.0 - 2026-06-15

- Parse MDict 2.x MDX dictionaries and MDD resource archives.
- Support uncompressed, LZO1X, and zlib-compressed blocks.
- Validate checksums, declared sizes, offsets, and configurable parser limits.
- Apply `KeyCaseSensitive` and `StripKey` semantics consistently.
- Add configurable loading, indexing, and decompressed-block caching.
- Add Swift 6 concurrency safety and asynchronous file opening.
- Add a separate SwiftUI dictionary-browser example package.
